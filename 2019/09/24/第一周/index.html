Binwalk工具
分析：binwalk filename 查看文件类型有可能是0x0-0x123之间是图片，之后是压缩包就要用到分离
分离：binwalk -e filename
Binwalk分离不出时用foremost 文件名 -o 输出目录名
dd 当文件自动分离出错或其他原因无法分离时
dd if=源文件 of=目标文件名 bs =1 skip =开始分离的字节数
if =file #输入文件名
of=fli e #输出文件名
bs=bytes #读写大小为btyes
skip=blocks #输入文件开口跳过blocks个块之后开始复制
例子：
1.txt
123456789abcdefg
dd if=1.txt of 2.txt bs=5 count=1
保存前5块另存为2.text
dd if 1.txt of =4.txt bs=5 count=3 skip=1(跳过第一块)

##Linux
基本命令
pwd:显示当前所在的工作目录
cd root 进入root目录
cd . 验证 ‘.’ 待代表当前目录
cd .. 返回上一层
cd ~回到刚才目录
ls ：显示当前目录文件
-a ：显示所有文件
- l ：显示所有文件、目录信息
-d ：显示当前目录本身属性（与 l 配合使用）
- i ：显示文件或目录的indoes（配合 l 使用）

  1. 创建、删除、修改
    touch file 创建文件file #当文件存在时，更新创建时间
    mkdir dir 创建目录dir
    rm [option]
    [option] :
     - i        确认信息，root用户默认有，普通用户无
- r     递归recursion 删除目录必须使用
- f     强制删除，没有提示和确认
-rf     组合使用
    mv [option] file dst 移动、重命名文件夹或者目录
    cp [option] scr dst 复制文件或者目录
    -p 保留原来的时间截，拥有者信息
-r 复制目录
  2. 文件查看操作
    cat 显示文件所有内容
    more 查看文件内容（只能往下）
    less 分页显示文件内容
    head、tail 显示文件首尾行
  3. vi编辑器——初级
    Vi 文件名 ：光标到字母输入相同字母进入编辑
    按Esc退出，按： q退出 w保存
：wq 保存退出
dd 删除当前行
  4. 网络地址配置
    ifconfig 网络配置命令
    不带参数 查看所有网卡信息 ————相当于Windows中的ipconfig → inet 192.
  5. 1.1
    Ifconfig ethN 查看网卡N 的信息
    Ifconfig ethN up/down，ifdown/ifup ethN 启用/禁用网卡N,断开、链接
    Ifconfig ethN 192.168.1.1 netmask 255.255.255.128 为网卡N配置ip地址
    重启网络服务，配置丢失，一边写进ifcfg-ethN文件中
    Dhclient（重新获取ip地址）
    -r 此参数释放地址
  6. 文件传输&shell
    Shell：
    使用工具securecrt/xshell
    Linux需要启用sshd
    修改配置文件vi /etc/ssh/sshd_congfig
    Add permitrootiogin yes
    Add PasswordAuthentication yes
    重启ssd服务
    /etc/init.d/ssh restart
    添加开机启动
    Update-rc.d ssh enable
    文件传输
    securecrt/xshell
    连接sftp会话
    Get：下载
    Put：上传
  7. 软件安装形式
    Apt-get 包管理工具
    -install 联网安装软件
    -remove 卸载
    源码安装流程
    ./configure 进行软件配置
    Make 进行编译
    Make install 进行安装
